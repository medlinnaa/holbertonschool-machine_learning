# Calculus

> Implementation and theoretical evaluation of foundational calculus operations for machine learning mathematics.


## 📖Description

This project provides a programmatic and theoretical approach to fundamental mathematical operations required for understanding machine learning algorithms. 

Rather than relying on external mathematical solvers, the project tasks involve writing Python functions from scratch to evaluate series operations and polynomial calculations. It bridges the gap between written mathematical notation and programmatic execution, ensuring a robust foundation in calculating rates of change, evaluating function areas, and manipulating lists representing mathematical polynomials without the use of standard iterative loops where forbidden.


## 🧠Concepts

* **Summation and Notation:** Decoding Sigma ($\Sigma$) and Pi ($\Pi$) notations to compute finite mathematical series and products, including factorials.
* **Differentiation Rules:** Utilizing the power, product, and chain rules to find the derivatives of polynomials and logarithmic functions.
* **Multivariable Calculus:** Taking partial derivatives of functions with multiple variables by isolating variables to compute gradients.
* **Integration:** Calculating both definite and indefinite integrals, including double integration over specific intervals and finding the antiderivative of multivariable functions.


## ⚙️Requirements

* Python 3.x
* Ubuntu 20.04 LTS
* Executable files must be strictly non-looping where specified (e.g., recursive or formulaic approaches).
* PEP 8 styling compliance for all Python scripts.
