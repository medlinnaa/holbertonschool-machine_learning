# Plotting

> Introduction to plotting and data visualization using Python's Matplotlib and NumPy libraries.


## 📖Description

This project serves as a practical introduction to generating visual representations of data. 

Rather than just calculating numbers and printing them to a console, this repository demonstrates how to transform raw datasets—such as statistical distributions, multivariate data, or mathematical models—into clear, readable graphics. By utilizing standard industry tools, the scripts provide a functional foundation for building analytical dashboards, comparing trends, and communicating data-driven insights effectively to an audience.


## 🧠Concepts

* **Basic Geometry & Plotting:** Mapping mathematical functions (like cubic formulas) and multivariate data points onto a 2D Cartesian plane using custom markers, line styles, and colors.
* **Axis Transformation:** Applying logarithmic scales to the Y-axis to visualize exponential behavior (like radioactive decay) more effectively, alongside precise control over axis limits and tick marks.
* **Data Binning:** Grouping continuous, normally distributed datasets into discrete numerical intervals to construct frequency histograms.
* **Figure Composition:** Utilizing subplot grids to arrange multiple distinct charts within a single, unified figure. This includes handling complex layouts where a single graph spans multiple columns, and adjusting spacing via tight layouts.
* **Categorical Stacking:** Building layered (stacked) bar charts to represent multi-dimensional categorical data by iteratively calculating and adjusting the baseline height for each new data series.


## ⚙️Requirements

* Python 3.9 (or higher)
* NumPy
* Matplotlib
* Ubuntu 20.04
