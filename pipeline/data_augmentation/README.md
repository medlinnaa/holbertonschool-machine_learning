# Data Augmentation (Image Preprocessing)

> Implementation of basic image data augmentation techniques using TensorFlow.

---

## 📖 Description

This project implements several **data augmentation operations** used in deep learning pipelines to artificially increase dataset diversity and improve model generalization.

The transformations include:

* Horizontal flipping
* Random cropping
* Rotation
* Contrast adjustment
* Brightness adjustment
* Hue modification

All operations are implemented using **TensorFlow image processing utilities**.

---

## 🧠 Concepts

* Data augmentation helps reduce overfitting by generating modified versions of images.
* Works directly on tensors (`tf.Tensor`).
* Common techniques used in computer vision pipelines.
* Helps models generalize better to unseen data.

---
