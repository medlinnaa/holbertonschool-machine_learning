# Convolutional Neural Networks (CNNs)

> Implementation of core CNN operations including forward/backward propagation, pooling layers, and a LeNet-5 style model using NumPy and TensorFlow/Keras.

---

## 📖 Description

This project implements fundamental components of **Convolutional Neural Networks (CNNs)**, covering both low-level operations and a full model architecture.

It includes:

* Convolutional layer forward propagation
* Pooling layer forward propagation
* Convolutional layer backward propagation
* Pooling layer backward propagation
* A modified **LeNet-5** model using Keras

These components form the basis of deep learning models used in computer vision tasks.

---

## 🧠 Concepts

* Convolution operation: feature extraction using kernels
* Padding: `"same"` vs `"valid"` convolution
* Stride: step size of filter movement
* Pooling:

  * Max pooling → strongest feature selection
  * Average pooling → feature smoothing
* Backpropagation:

  * Gradient flow through convolution and pooling layers
* Fully connected layers for classification
